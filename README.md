The LINGO files provided in this repository contain the optimization models used in the Dissertation entitled "Development of Inoperability Assessment and Optimization Framework for an Integrated Plastic Recycling Network". Running these models requires access to LINGO software, which may necessitate a valid license.

Using the Model:
The file can only be run if you have LINGO software installed. To execute it, open the file into the LINGO software, go to the Solver tab, and then click Solve. A new tab will appear showing the optimization results.

Modifying the Data:

Modifying the Model:
The bottom section of each file contains the optimization model. A brief description is provided before each equation. You can modify these equations if you wish to further improve or extend the model. For easier modification, it is recommended to open the file in the LINGO software.
